# Code of Conduct

As part of the Cortlet software brand, this project follows the official Cortlet Code of Conduct.

We are committed to providing a safe, respectful, and productive environment for everyone. By participating in this project, you agree to abide by our standards regarding:

1. **Expected Behavior**: Being respectful, constructive, and professional.
2. **Unacceptable Behavior**: Harassment, hateful conduct, or intentional abuse of software/APIs.
3. **Reporting**: Violations should be reported privately to support@cortlet.com.

For the full, legally binding text, please visit:
https://legal.cortlet.com/code-of-conduct
